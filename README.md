## 🧭 Project Overview

This repository is my **Digital Literacy Portfolio**, submitted as part of the CSE0001 course at VIT Bhopal University. The project is structured around the scenario of being selected as a **Student Digital Ambassador** — someone who helps peers navigate the digital world responsibly and professionally.

The portfolio covers five core modules of the course, each mapped to a hands-on task:

| # | Task | Module | Marks |
|---|------|--------|-------|
| 1 | Digital Literacy Awareness Infographic | Module 1 | 20 |
| 2 | Student Digital Portfolio | Module 2 | 20 |
| 3 | Coding & Collaboration Platforms | Module 3 | 20 |
| 4 | Professional Email & Etiquette Guide | Module 4 | 20 |
| 5 | Cybercrime Awareness Case Study | Module 5 | 20 |

**Total: 100 marks | Pass mark: 50**

> No prior coding knowledge is required. All tools used are free and browser-based.
